# Быстрый деплой на Vercel

1. Создать новый репозиторий на GitHub.
2. Загрузить все файлы из этой папки в репозиторий.
3. Зайти в Vercel.
4. Add New -> Project.
5. Import Git Repository.
6. Выбрать репозиторий.
7. Framework Preset: Vite.
8. Build Command: npm run build.
9. Output Directory: dist.
10. Deploy.

API-ключи не нужны: в этой версии работает бесплатный локальный AI-разбор команд прямо в браузере.

## Как обновлять

После изменений загружайте новые файлы в тот же GitHub-репозиторий. Vercel автоматически сделает новый deploy. После этого на телефоне достаточно обновить страницу.
