# Этапы по блокноту

Обновлено: 2026-05-08

## Уже сделано

- Антидубль голосовой команды: одинаковая фраза не должна повторно отрабатывать в коротком окне.
- Нумерация записей в папках и в общем списке.
- Удаление записи по номеру голосом: `удали запись 2`, `удали список 3 из папки покупки`.
- Копирование записи в буфер обмена голосом и кнопкой.
- Базовый календарь на 5 лет вперёд с выбором даты и сохранением записи.
- Автосохранение календарной записи в блокнот как appointment.
- Прод-деплой на Vercel и рабочий архив на рабочем столе.

## Сделано в текущем проходе

- Возвращена защита от дублей сохранения в коротком окне без блокировки нормального дальнейшего добавления.
- Смысловой поиск усилен простыми синонимами: `мастер`, `гараж`, `аптека`, `работа`, `стрижка`, `приложение`.
- Напоминания в настройках синхронизируются с календарной формой.
- Первое и второе напоминание можно включать/отключать отдельно.
- Голосовые команды для напоминаний обновляют текущую calendar-note, выбранную дату или глобальные настройки.
- Открытие calendar note подгружает её время и напоминания обратно в календарь.
- Вкладки `календарь` и `настройки` теперь принудительно взаимоисключающие на уровне state.
- Голосовая работа с уже выбранной датой стала контекстной: `открой 10 сентября`, `запиши туда ...`, `оставь напоминание ...`.
- Если на выбранной дате уже есть calendar-запись, команда открытия даты подтягивает её в календарную форму для дальнейшего редактирования.
- Добавлено голосовое обновление существующей calendar-записи на выбранной дате: `измени на эту дату ...`, `обнови запись туда ...`.
- Календарь переведён с вертикальной ленты на горизонтальную прокрутку по месяцам.
- Горизонтальный календарь выровнен по snap: месяцы должны останавливаться ровно в рамке.
- 1-е и 2-е напоминание теперь включаются и выключаются отдельно и в настройках, и в календаре.
- Календарные голосовые даты теперь понимают и обратный порядок месяца: `октябрь 7 число`, `сентябрь 10 число`.
- Улучшен разбор фраз вида `добавь на 7 сентября на 8 вечера напоминание позвонить другу`.
- Дни с записями в календаре подсвечиваются заметнее и показывают количество записей на дату.
- Shopping-правила вынесены в `src/lib/notebookRules.js`, чтобы уменьшить размер контекста для следующих сессий.
- Timed-покупки теперь ставят первое уведомление прямо на время покупки, а не на утренний дефолт.
- Date/reminder voice-парсинг вынесен в `src/lib/voiceCalendar.js`.
- Добавлен быстрый smoke-прогон `npm run smoke:voice` на реальные фразы пользователя: `7 мая мне на стрижку`, timed-shopping, второе напоминание, открытие даты, очистка calendar-фразы.
- Усилен антидубль: теперь режутся не только дословные, но и почти одинаковые голосовые повторы в коротком окне.
- Быстрые даты под статусом переведены в отдельный горизонтальный scroll-контейнер.
- Напоминания получили resume-проверку после возврата в открытую вкладку, чтобы не терять ближайшее уведомление при фоне/сне браузера.
- Добавлен Web Push слой для закрытой страницы: `api/push-config.js`, `api/reminders-sync.js`, `api/reminders-dispatch.js`.
- Подключено серверное хранилище расписаний через Vercel Blob (`BLOB_READ_WRITE_TOKEN`) с fallback на Upstash Redis REST.
- Vercel Hobby заблокировал минутный cron, поэтому dispatcher вынесен в GitHub Actions workflow `.github/workflows/reminders-dispatch.yml`.
- Service worker теперь принимает push-события и показывает уведомления в верхнюю шторку телефона.
- Production env Vercel уже содержит `VAPID_PUBLIC_KEY`, `VAPID_PRIVATE_KEY`, `VAPID_SUBJECT`, `BLOB_READ_WRITE_TOKEN`.
- После сохранения календарной записи серверное расписание синхронизируется сразу, без ожидания фонового `useEffect`.
- Добавлен диагностический endpoint `/api/reminders-status`: показывает количество подписок, pending/due reminders и ближайшее время без вывода приватных push-ключей.

## Важные файлы

- `src/App.jsx` — основная логика блокнота, календаря, поиска, голосовых команд и уведомлений.
- `api/reminders-sync.js` — сохраняет push-подписку телефона и расписание напоминаний.
- `api/reminders-dispatch.js` — cron-диспетчер серверных push-уведомлений.
- `api/reminders-status.js` — безопасная диагностика серверного расписания.
- `api/_push-store.js` — Vercel Blob/Redis хранилище расписаний.
- `public/sw.js` — service worker, локальные таймеры и обработка push-событий.
- `.github/workflows/reminders-dispatch.yml` — внешний scheduler для Vercel Hobby.
- `src/lib/voiceCalendar.js` — вынос дат, времени, calendar-target и reminder voice-парсинга.
- `scripts/voice-smoke.mjs` — сценарная проверка ключевых голосовых фраз.
- `TEST_PLAN.md` — сценарии ручной проверки.
- `README.md` — пользовательская документация.
