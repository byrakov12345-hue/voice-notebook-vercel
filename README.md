# Умный голосовой блокнот

Готовый проект для Vercel с бесплатным встроенным локальным AI-разбором команд.

Основные функции:

- голосовые команды на русском;
- локальный AI-парсер команд без API-ключа и без платного сервера;
- автопапки: Идеи, Встречи, Покупки, Контакты, Коды, Задачи и другие;
- голосовой поиск;
- контакты с кнопками позвонить/написать;
- поделиться, копировать, удалить с подтверждением;
- корзина и восстановление;
- хранение в localStorage.

## Как работает локальный AI-разбор

Приложение не отправляет команды на внешний AI по умолчанию. Оно разбирает фразы прямо в браузере:

- определяет намерение: сохранить, найти, показать, удалить, позвонить, написать, поделиться;
- определяет тип записи: идея, встреча, покупка, контакт, код, задача, расход, заметка;
- вытаскивает данные: имя, номер телефона, список товаров, время встречи, код;
- выбирает папку;
- просит подтверждение для удаления.

## Локальный запуск

1. Установить зависимости: `npm install`
2. Запустить: `npm run dev`

## Vercel

1. Загрузить проект в GitHub.
2. Import Project в Vercel.
3. Framework Preset: Vite.
4. Build Command: `npm run build`.
5. Output Directory: `dist`.
6. Deploy.

Технически:

- проект закреплён под `Node 22.x`;
- lockfile из локального агентского окружения не хранится, чтобы Vercel ставил зависимости из публичного npm без конфликтов registry.

OPENAI_API_KEY не нужен.

## Как обновлять

После изменений загружайте новые файлы в тот же GitHub-репозиторий. Vercel автоматически сделает новый deploy. После этого на телефоне достаточно обновить страницу.
