# План проверки на телефоне

## Быстрая проверка после распаковки

```bash
npm ci
npm run smoke:voice
npm run build
npm run dev -- --host 0.0.0.0
```

На телефоне в той же сети откройте `http://IP_КОМПЬЮТЕРА:5173`. Для шторки уведомлений после закрытия браузера используйте HTTPS/Vercel-деплой.

## Команды для ручного поля и микрофона

1. `У меня идея хочу вечером в душ` — папка Идеи.
2. `Мне завтра на стрижку в 8 вечера` — папка Встречи, время 20:00, календарная запись.
3. `Запомни список покупок: молоко, хлеб, яйца` — папка Покупки, список.
4. `Запомни номер телефона Сергей гараж 050 123 45 67` — контакт с кнопками позвонить/написать.
5. `Найди номер Сергея гараж` — открывается контакт.
6. `Запомни комбинацию цифр один два три четыре и выведи на экран` — папка Коды и комбинации, значение 1234.
7. `Открой календарь` — календарь появляется справа.
8. `Открой 10 число сентября и запиши стрижка в 11 вечера` — запись в календаре справа и в папке Встречи.
9. `Поставь второе напоминание на 8 вечера` — у выбранной календарной записи появляется второе уведомление.
10. `Включи уведомления` — браузер запрашивает разрешение.
11. `Покажи последнюю заметку` — открывается последняя запись.
12. `Удали запись 1` — удаляется запись под номером 1 из текущего списка.

## Визуальная проверка

- На ПК: слева функции/папки, по центру записи, справа микрофон и календарь.
- На телефоне: блоки идут сверху вниз, без горизонтального сдвига страницы.
- Календарь листается горизонтально внутри своей карточки, а не двигает всю страницу.
- Длинные названия папок и записей обрезаются/переносятся внутри карточек.

## Проверка уведомлений

1. Откройте HTTPS-версию.
2. Нажмите `Проверить шторку` или скажите `включи уведомления`.
3. Разрешите уведомления.
4. Создайте запись на ближайшее время: `напомни через ...` или календарную запись с временем.
5. Проверьте, что service worker получил расписание: в UI поле `Память телефона` должно показать время синхронизации.
6. Для полной фоновой проверки нужен Web Push: Vercel env + GitHub Actions dispatcher.
